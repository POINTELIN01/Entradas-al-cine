# Entradas al cine

Este proyecto es un programa sencillo desarrollado en Python que valida si una persona puede ingresar al cine con base en si cuenta o no con un boleto.

## Descripción
El programa solicita al usuario que indique si tiene boleto ingresando una respuesta de tipo "si" o "no".  
Con base en la respuesta, el sistema permite o niega el acceso, además de validar que la entrada sea correcta.

## ¿Cómo funciona?
1. El programa pregunta al usuario si tiene boleto.
2. Si la respuesta es "si", se muestra un mensaje indicando que puede entrar.
3. Si la respuesta es "no", se muestra un mensaje indicando que no puede entrar.
4. Si la respuesta no es válida, el programa solicita una entrada correcta.
